# vapourware.ai

KJV Bible reader inspired by yews.news.

- Mobile-first, 260px column, light gray bulletin aesthetic
- Infinite swipe through all 1,189 chapters (Genesis 1 → Revelation 22)
- Tap header to toggle book list, tap a book to jump
- Chapters fetched on demand from [dailybible.ca](https://dailybible.ca) API
- 3 files, no build step, no dependencies beyond Express for serving
